Breve explicação
O site foi feita de forma simples, com o foco apenas na área em que os cálculos fossem utilizados. A cor azul mantém a neutralidade, mas ainda destaca melhor que um fundo cinza ou completamente branco. Os detalhes em círculos ou várias linhas formando um quadrado foram intencionais, causando um certo interesse, mesmo que mínimo para o usuário.
